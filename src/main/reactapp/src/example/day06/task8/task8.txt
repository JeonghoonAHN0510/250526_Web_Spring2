실습8 : React(+ Router + Redux + Persist)

1. 제품 페이지, 장바구니 페이지 구현
    제품 : 특정 제품의 [담기] 버튼을 클릭하면, 전역 상태에 저장한다.
    장바구니 : 전역 상태에 저장된 제품들의 목록을 출력한다.
        -> 제품명 + 수량 + 총 가격


- pages : page 컴포넌트들을 저장하는 폴더
    -> HomePage.jsx
    -> MenuPage.jsx
    -> CartPage.jsx
- store : redux관련 store와 slice를 저장하는 폴더(컴포넌트가 아니므로 소문자 시작)
    -> cartSlice.jsx
    -> store.jsx
- components : 컴포넌트들을 저장하는 폴더
    -> Header.jsx
- App.jsx : 최초 main.jsx에서 렌더링되는 컴포넌트

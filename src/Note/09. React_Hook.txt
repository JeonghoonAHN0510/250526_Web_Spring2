- DOM : Document Object Model
    1. HTML 문서의 구조를 표현하는 객체
    2. Element : HTML 마크업들, <div> <input> <p>
    3. Attribute : HTML 마크업들의 속성, style id class src href

- React DOM : Virtual Document Object Model
    1. 실제 DOM을 직접 수정하지 않고 가상 DOM을 생성하여 수정한다.
    2. Element : 컴포넌트
    3. Attribute : props

[ 생명주기 ] Life Cycle
- 컴포넌트의 생성 시점부터 종료 시점까지의 주기
- 주기(시점)
    1. mount : 출생, main.jsx가 컴포넌트를 그려주는 함수(?)
        -> 상위컴포넌트가 return할 때, 하위컴포넌트도 return하므로 하위컴포넌트도 출생한다.
        -> 컴포넌트가 출생될 때, 한 번 실행되는 함수 : useEffect( () => {}, [ ] )

    2. update : 인생, 컴포넌트가 재실행되는 함수
        -> 컴포넌트가 재실행되므로 리렌더링된다.
        -> 컴포넌트가 수정될 때마다 실행되는 함수 : useEffect( () => {}, [ 의존성배열 ] )
        -> 의존성 배열 : 특정한 변수를 넣어, 특정한 변수가 변경될 때 실행

    3. unmount : 사망, 컴포넌트가 사라질 때 -> 자동

[ Hook ]
- **React의 핵심**
- 컴포넌트와 생명주기를 기준으로 원하는 시점에 정해진 함수를 실행하도록 돕는 함수
- 종류
    1. useState
    2. useEffect
